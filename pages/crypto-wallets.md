## Adding Crypto Wallets

### Generate New Wallet

1. **Access Crypto Section**
   - Log in to dashboard
   - Click "Crypto Wallets"

2. **Create Wallet**
   - Select cryptocurrency
   - Choose wallet type (hot/cold)

3. **Secure Wallet**
   - Generate seed phrase
   - Set up 2FA
   - Download backup

### Import Existing Wallet

1. **Import Option**
   - Select "Import Wallet"
   - Choose import method

2. **Provide Details**
   - Private key (not recommended)
   - Seed phrase (recommended)
   - Hardware wallet connection

3. **Verify Import**
   - Confirm balance
   - Test small transaction

## Crypto Transactions

### Deposits

#### Deposit Process
1. Generate deposit address
2. Send crypto from external wallet
3. Wait for confirmations
4. Funds credited to account

#### Deposit Requirements
- Minimum deposit: 0.0001 BTC equivalent
- Network confirmations: 1-6 (crypto-dependent)
- No deposit fees (network fees apply)

### Withdrawals

#### Withdrawal Process
1. Select withdrawal amount
2. Enter destination address
3. Confirm transaction details
4. 2FA verification required

#### Withdrawal Limits
- Daily: $10,000 (basic), $100,000 (verified)
- Network fees: Market rate
- Platform fee: 0.1%
